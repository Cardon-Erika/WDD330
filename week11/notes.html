Week Eleven Notes

This week I continued developing my final project. My big tasks for the week were streamlining some of the code, allowing players to be listed on a selected team, and implementing a favorite team bar.

I found that many of the teams in my api did not have individual player information, so I also had to create some code to catch if there was nothing returned and let the user know there was nothing to display for that team.

The favorites bar is still pretty crude. I have the main elements there, but I still need to add a few more team statistics to the bar, decide on the layout, and most importantly implement the local storage.
